# Image-generator-gen-ai
This project focuses on building an AI-powered image generation system that can create realistic or artistic images from text descriptions using Generative AI models.
It leverages advanced deep learning techniques like diffusion models and transformers to convert natural language prompts into high-quality images.

# Technologies Used
  Generative AI
  Diffusion Models
  PyTorch 
  Stable Diffusion 
  Python
